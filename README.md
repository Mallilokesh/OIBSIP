# OIBSIP
# 🚀 AI Universe Landing Page

A modern and responsive AI-themed landing page showcasing popular AI companies, trending AI tools, and the future of Artificial Intelligence. Built using HTML and CSS with a futuristic cyber-inspired design.

## 🌐 Live Demo

🔗 Live Website: https://mallilokesh.github.io/OIBSIP/

## 📌 Features

- Modern AI Startup Style UI
- Fully Responsive Design
- Futuristic Dark Theme
- Neon Glow Effects
- Glassmorphism Cards
- Animated Gradient Hero Section
- Trending AI Tools Section
- AI Companies Showcase
- Smooth Hover Animations
- Clean and User-Friendly Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Web Design
